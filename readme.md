# ☕ BaristaFlow: Akıllı Envanter Yönetimi

BaristaFlow, kafeler için tasarlanmış, stok takibi yapan ve kritik seviyelerde **Telegram** üzerinden bildirim gönderen modern bir web uygulamasıdır.

## 🔥 Özellikler
- 📊 **Dinamik Grafikler:** Kategori ve stok dağılımını anlık izleme.
- 🤖 **Telegram Botu:** Stok azaldığında cebine bildirim gelsin.
- 📸 **Görsel Takip:** Ürün fotoğrafları ile kolay yönetim.
- 🌙 **Dark Mode:** Göz yormayan modern tasarım.

## 🛠️ Teknolojiler
- **Frontend:** React, TypeScript, Chart.js, Lucide Icons.
- **Backend:** Node.js, Express, Multer, Telegram Bot API.
- **Database:** MySQL.

## 🚀 Kurulum
1. `npm install` (Hem frontend hem backend için).
2. `.env` dosyasını oluştur ve API anahtarlarını gir.
3. `npm run dev` ile sistemi başlat.
